do this before starting to code for versioning and stuff

1. install docker desktop if you haven't
2. install node version 20, run
    - nvm install
    - nvm use


docker compose -f infra/docker-compose.yml up -d
cd backend
npm install
cp .env.example .env
npx prisma migrate dev



running website:

run backend first:
cd backend
npm install
cp .env.example .env (if you haven't already)
npx prisma migrate dev
npm run dev


run frontend:
cd frontend
npm install
npm run dev

if you want to view the database:
npx prisma studio
